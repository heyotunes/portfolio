PROJECT OVERVIEW

CRYPTOLAND

Cryptoland is a website that allows you to track the value of different cryptocurrencies in real-time. This way, you can make informed decisions about when to buy or sell your coins. I built this app for a startup company called Abyiss, with so many different coins on the market, it can be hard to keep track of the value of each one. That's where a crypto price app comes in handy.
