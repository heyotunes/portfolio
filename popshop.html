PROJECT OVERVIEW

POPSHOP

PopShop is a simple react project i created with javascript and typescript. This website allows you to manage perfomance grades in an organization. With this system you can input numbers, store it in a database and display it in an ascending order on a seperate page. The process was very simple. I started by creating a site with an input field on the page, then I created a script that would detect when someone clicked on the input field and store their score in a firebase-database. When they clicked submit, it would display their score on the page as well as their name./p>
